数据类型

Java是一种强类型语言。意思就是每一个变量都需声明一种数据类型。

在Java中，一共有8种基本类型（primitive type），其中有4种整型、2种浮点类型、1种用于表示Unicode编码的字符单元的字符类型char和1种用于表示真值的boolean类型。

类型 存储需求 取值范围
byte 1字节 = 8bit -128 ~ 127
short 2字节 = 16bit -32768 ~ 32767
int 4字节 = 32bit -2147483648 ~ 2147483647
long 8字节 = 64bit -9 223 372 036 854 775 808
~
9 223 372 036 854 775 807
float 4字节 = 32bit 大约±3.402 823 47E+38F
（有效位数为6 ~ 7位）
double 8字节 = 64bit 大约±1.797 693 134 862 315 70E+308
（有效位数为15位）
char 2字节 = 16bit Unicode 字符
boolean 0.125字节 = 1bit true、false

解释：1字节 = 8bit
