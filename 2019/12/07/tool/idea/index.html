第二步：安装IDEA，放置好补丁

安装IDEA, 将补丁 jetbrains-agent.jar，并把该jar包放到一个文件夹中，建议放到IDEA的安装目录bin文件夹下。

第三步：配置补丁位置

打开安装好的IDEA ，选择菜单栏中的 Help -> Edit Custom VM Options，配置补丁位置，格式为，-javaagent:D:\software\idea\creack_package\jetbrains-agent.jar，其中补丁jetbrains-agent.jar按自己的实际位置填，操作如下图：
