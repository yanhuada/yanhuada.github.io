发表于 2026-04-30 分类于 python

一、str — 用户友好的字符串表示

触发时机

  • 调用 str(obj) 时
  • 使用 print(obj) 时
  • 使用 f”{obj}” 或 “{}”.format(obj) 时

用途
给普通用户看的可读字符串，注重可读性

二、repr — 开发者调试用的字符串表示

触发时机

  • 在交互式解释器中直接输入对象时
  • 调用 repr(obj) 时
  • 未定义 str 时，print() 会回退使用 repr
  • f”{obj!r}” 时

用途
给开发者看的字符串，注重准确性，理想情况下可以用来重建对象。

三、len — 定义对象长度

触发时机

  • 调用 len(obj) 时
  • 用于布尔判断时（len == 0 则为 False）
